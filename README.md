## üöÄ How to Run These Scripts

From the terminal inside the repository:

### Ticket Summarizer
```
python support-tools/summarize_ticket.py
```

### CSV Cleaner  
*(ensure sample.csv is inside the data-cleaning folder)*  
```
cd data-cleaning
python clean_csv.py
```

### API Caller  
```
python api-examples/call_api.py
```

These scripts run in any Python 3.9+ environment, including GitHub Codespaces.
